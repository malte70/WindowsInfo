WindowsInfo

  • Github Project
  • Download

WindowsInfo is a small utility which shows you some details about your Windows Operating system. Since 32 Bit applications running on a 64 Bit Windows don't see the same Registry values as 64 Bit applications, you need to download and run the correct WindowsInfo version for your Processor architecture. All Windows versions since Windows XP are supported.
