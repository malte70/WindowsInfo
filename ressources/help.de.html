WindowsInfo Hilfe

WindowsInfo zeigt nützliche Informationen über ein Windows-System an.

Grafische Oberfläche

Die meisten Nutzer werden wohl die grafische Oberfläche benutzen. Sollten während der Installation keine Verknüpfungen auf dem Desktop oder anderswo angelegt worden sein, muss die Datei WindowsInfo.exe aus dem Programmverzeichnis (C:\Programme unter Windows XP, seit Windows Vista C:\Program Files) per Doppelklick gestartet werden.

WindowsInfo schließen

Geschlossen wird das Fenster entweder über den Schließen-Knopf oder den Menüeintrag Datei→Beenden

Anzeigeformat wählen

Beim Start zeigt WindowsInfo Die Informationen in einem übersichtlichen Format an, wie es auch in Version 0.1.2 der Fall war. Über den Menüpunkt Anzeige→Details kann die detailierte Ansicht gewählt werden, die der Ausgabe von WindowsInfo.CLI.exe entspricht.
