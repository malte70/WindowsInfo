WindowsInfo Help

WindowsInfo shows some useful information about a Windows system.

Graphical User Interface

Most users will use the GUI. if there wasn't made any shortcut during the installation, you must double-click on the file WindowsInfo.exe located at the application directory (C:\Program Files\Windows Info).

Close WindowsInfo

To clode the application window, either click on on the Close button of the window or select File→Exit.

Change view

Per default, WindowsInfo shows the information in a neat format, like in Version 0.1.2. By selecting on View→Detailed the detailed view can be selected. The information is shown just like WindowsInfo.CLI.exe would do.
